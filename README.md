# Domternal

A ProseMirror-based rich text editor framework.

> **Note:** This project is in early development.

## Packages

| Package | Description |
|---------|-------------|
| `@domternal/core` | Framework-agnostic ProseMirror editor engine |

## Development

```bash
# Install dependencies
pnpm install

# Build all packages
pnpm build

# Run tests
pnpm test

# Lint
pnpm lint

# Type check
pnpm typecheck
```

## Requirements

- Node.js >= 20
- pnpm >= 10
